# 更新日志

## [2026-01-02] v2.0 - 粘性增强大更新

### 新增功能

#### 1. 每日任务系统 (`/tasks`)
- 每天3个随机小任务，完成后自动发放 MP 奖励
- 任务类型：聊天、签到、塔罗、锻造、盲盒、决斗、转赠、购物
- VIP 用户奖励 +50%

#### 2. 幸运转盘 (`/wheel`)
- 每天免费转1次，VIP 用户2次
- 奖池：5-1000 MP、幸运草、防御卷轴、各种道具券
- 传说宝箱概率 0.5%（500-1000 MP）

#### 3. 在线活跃度系统 (`/active`)
- 群聊发言累积活跃点
- 达到阈值自动发奖：
  - 50点 → 25 MP
  - 150点 → 75 MP
  - 350点 → 175 MP
  - 700点 → 350 MP
  - 1200点 → 600 MP
- VIP 活跃获取翻倍

#### 4. 幸运空投 (`/airdrop`)
- 管理员可手动触发群空投
- 120秒内第一个点击的人获得
- 宝箱类型：青铜(30-80)、白银(60-150)、黄金(100-300)、钻石(200-500)、传说(500-1000)

#### 5. 统一任务系统 (`/missions`)
- 融合每日任务和悬赏任务
- 标签页切换，界面统一简洁

### 新增命令
| 命令 | 功能 |
|------|------|
| `/tasks` / `/missions` | 查看统一任务界面 |
| `/wheel` / `/spin` | 幸运转盘抽奖 |
| `/active` / `/presence` | 查看活跃度 |
| `/rank` | 活跃排行榜 |
| `/airdrop` | 触发幸运空投(管理员) |
| `/changelog` | 查看更新日志 |

### 界面更新
- 主菜单新增「每日任务」和「幸运转盘」入口
- 帮助文档更新，收录所有新功能
